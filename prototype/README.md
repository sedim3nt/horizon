# Horizon Prototype

This isolated stakeholder prototype demonstrates Horizon's public
newest-activity-first opportunity feed using fictional data. It is not connected
to Queen Raida, Prism, Discord, or the CRM.

The interface follows the
[Horizon PRD](https://github.com/sedim3nt/horizon/blob/main/documents/PRD.md)
and uses the
[RaidGuild brand system](https://github.com/raid-guild/brand/blob/main/BRAND-ASSETS.md).

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Generated source images are preserved in the
[repository gallery](https://github.com/sedim3nt/horizon/tree/main/gallery).
