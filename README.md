# Horizon

Horizon turns Queen Raida, meeting, Discord, and CRM context into a
newest-activity-first feed of RaidGuild business-development opportunities.

## Stakeholder Prototype

The
[interactive prototype](https://github.com/sedim3nt/horizon/tree/main/prototype)
uses fictional data to demonstrate the feed, filters, thread history, ownership,
next actions, and generated imagery.

```bash
cd prototype
npm install
npm run dev
```

Every generated image, including unused explorations, is preserved in the
[gallery](https://github.com/sedim3nt/horizon/tree/main/gallery).
