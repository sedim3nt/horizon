# Horizon

Horizon is a RaidGuild BD visibility tool. It turns Queen Raida's Discord,
meeting, and CRM context into a public-safe web surface for active opportunities,
project threads, and calls to action.

The first artifact in this repo is the product requirements document:

- [PRD.md](PRD.md)

## Source Direction

Horizon should use the operational AI tone from RaidGuild AI: signal, context,
coordination, automation, and practical workflows. The product should avoid
fantasy framing in the interface and focus on helping members quickly understand
what is active, where help is needed, and which opportunities are moving.
