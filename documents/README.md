# Horizon

Horizon turns Queen Raida, Prism, Discord, meeting, and CRM context into a
public, chronological feed of RaidGuild business-development threads.
Meaningful updates move a thread to the top and add a brand-consistent
illustration.

The implementation adapts the
[ClawRyderz knowledge-graph template](https://github.com/sedim3nt/telegram-knowledge-graph):
private source atoms, scheduled AI synthesis, generated Markdown, Quartz 4, and
static deployment.
